# xanderxx.xyz 🌐 ![GitHub Repo stars](https://img.shields.io/github/stars/Xanderxx46/about-me-page?style=social)

This is the source code for [xanderxx.xyz](https://xanderxx.xyz), the personal website and digital home of Xander Hogan. The site is built with React and styled using Tailwind CSS to deliver a fast, modern, and responsive user experience.

---

## 🔧 Built With

- [React](https://reactjs.org/) – Component-based UI library  
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework

---

## 🚀 Features

- 🧑‍🎨 **About Me** – Personal introduction and background  
- 📂 **Projects** – Showcase of creative and technical work  
- 📡 **External Links** – Central hub for socials and contact info  
- 🌑 **Dark Mode Support** – Modern design with optional dark theme  
- 📱 **Responsive Design** – Optimized for mobile, tablet, and desktop
